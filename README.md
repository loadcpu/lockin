# Lock In

Lock In is a strict macOS focus app that blocks distracting apps and websites during timed sessions.

Free. Local-first. No account. No subscription. No telemetry.

## Requirements

- macOS 13 or later

## Install

Download `LockIn.dmg` from the latest release, open it, and drag `Lock In.app` into `/Applications`.

## Build

```sh
./build.sh
```

## Test

