# Lock In

Lock In is a strict macOS app blocker.

It blocks apps and websites during a timed focus session.

Free. Local-first. No account. No subscription. No telemetry.

## Install

Download `LockIn.zip` from the latest release, unzip it, and move `Lock In.app` to `/Applications`.

Or install from Terminal:

```sh
curl -fsSL https://raw.githubusercontent.com/loadcpu/screen-blocker/main/install.sh | bash
```

Requires macOS 13 or later.

## Build

```sh
./install.sh
```
